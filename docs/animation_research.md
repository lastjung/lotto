## 추천 옵션

### 1순위: 로또볼 추첨기
- 실제 추첨 느낌
- 공 튕기는 사운드
- 긴장감 최고

### 2순위: AI 스캐너
```
[1단계] 🔬 1~45 번호가 빠르게 스캔
    ↓
[2단계] 🎯 번호 하나씩 "락인" 광채 효과
    ↓
[3단계] 🎉 완성 시 컨페티 + 사운드
```

### 3순위: 슬롯머신
- 클래식 카지노 느낌
- 구현 상대적 용이

## 기술 구현 방식

### CSS 애니메이션
- `transform: translateY()` - 슬롯 회전
- `transform: rotate()` - 룰렛 스핀
- `transition` - 부드러운 전환

### JavaScript
- `Math.random()` - 랜덤 결과
- GSAP 라이브러리 - 고급 애니메이션
- Web Audio API - 사운드 효과

### 사운드 효과 아이디어
- 🔊 공 섞이는 소리 (드럼롤)
- 🔊 공 튀어나오는 "퐁" 소리
- 🔊 세트 완성 시 "딩동" 효과음
- 🔊 배경음악 (선택적)
