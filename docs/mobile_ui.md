# 📱 모바일 & 태블릿 UI 최적화 가이드

> 마지막 업데이트: 2026-01-03

## 1. ✅ 완료된 작업

### 하단 탭 바 (Bottom Navigation)
- `lg:hidden` - 1024px 미만에서 하단에 4개 탭 표시
- Dashboard, History, Models, System
- 탭 전환 시 활성 버튼 하이라이트

### 사이드바 브레이크포인트 변경
- `hidden lg:flex` - 1024px 이상에서만 사이드바 표시
- 모바일/태블릿에서는 하단 탭으로 네비게이션

### 차트 숨김 (Mobile/Tablet)
- `hidden lg:grid` - 차트 2개 데스크탑에서만 표시
- 모바일/태블릿에서 화면 공간 확보

### 결과 화면 최상단 배치
- 모바일에서 생성된 볼(Results)이 가장 먼저 표시
- AI Engine(모델 선택)은 그 아래 배치
